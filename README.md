# 📚 Custom Generic List Library (C#)

## 📌 Описание проекта

Данный проект представляет собой реализацию **обобщённой структуры списка** на языке C# с поддержкой двух различных реализаций хранения данных:

- 🔹 `ArrayList<T>` — реализация на основе массива  
- 🔹 `LinkedList<T>` — реализация на основе узлов (Node)

Проект включает:

- Собственный интерфейс `IList<T>`
- Утилитарный класс `ListUtils` с расширенными методами
- Windows Forms UI для тестирования всех операций
- Поддержку типов:
  - `int`
  - `string`
  - `MyClass`

# 🗂 Реализация хранения

## 🔹 ArrayList<T>

- Основан на динамическом массиве `T[]`
- Быстрый доступ по индексу — **O(1)**
- Вставка и удаление требуют сдвига элементов
- Реализован без использования `System.Collections.Generic.List<T>`

### ✅ Преимущества:

- Высокая производительность при чтении
- Компактное хранение данных
- Удобство сортировки

---

## 🔹 LinkedList<T>

- Основан на классе `Node<T>`
- Каждый элемент хранит ссылку на следующий
- Нет перераспределения массива
- Более гибкая структура

### ✅ Преимущества:

- Удобная вставка и удаление
- Отсутствие копирования массива
- Демонстрация альтернативной структуры данных

---

# 🛠 ListUtils — дополнительные методы

Класс `ListUtils` реализует расширенную функциональность:

| Метод | Назначение |
|-------|------------|
| `Sort` | Сортировка списка |
| `Exists` | Проверка существования элемента по условию |
| `Find` | Поиск первого подходящего элемента |
| `FindLast` | Поиск последнего подходящего элемента |
| `FindIndex` | Индекс первого найденного |
| `FindLastIndex` | Индекс последнего найденного |
| `FindAll` | Возвращает все элементы, удовлетворяющие условию |
| `ConvertAll` | Преобразование списка в другой тип |
| `ForEach` | Выполнение действия для каждого элемента |
| `CheckForAll` | Проверка, удовлетворяют ли все элементы условию |

# 🖥 Пользовательский интерфейс

Проект содержит **Windows Forms интерфейс** для тестирования всех операций.

## 📥 Поля ввода

```csharp
inputBox  // значение элемента
indexBox  // индекс для операций
fromBox   // начало диапазона SubList
toBox     // конец диапазона SubList
```

### 📌 Назначение полей:

- **inputBox** — ввод значения элемента  
- **indexBox** — индекс для `Insert` и `RemoveAt`  
- **fromBox** — начальный индекс для `SubList`  
- **toBox** — конечный индекс для `SubList`

```csharp
public class MyClass : IComparable<MyClass>
{
    public int Number { get; set; }

    public int CompareTo(MyClass other)
    {
        return Number.CompareTo(other.Number);
    }

    public override string ToString()
    {
        return Number.ToString();
    }
}
```
